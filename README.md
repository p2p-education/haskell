
# Usage 


## Install Haskell Platform

## Install git

## Download this repo and build

~~~
$ git clone http://www.github.com/p2p-education/haskell
$ cd haskell && cd GL 
$ make clean && ghc arrow.hs
~~~

## execution

just type

~~~
$ ./arrow
~~~

if you are using Windows, execute 

~~~
arrow.exe
~~~
